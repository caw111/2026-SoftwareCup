# AI 学习陪练设计规范

> 版本：1.0  
> 适用范围：本项目全部桌面端与移动端页面  
> 视觉关键词：温暖、克制、清晰、轻量、学习陪伴感

## 1. 使用原则

- 新建或重构页面前，必须先阅读本文件，并优先复用这里定义的设计令牌和组件规则。
- 页面以高信息密度的学习工作台为主，不使用营销页式巨型标题、装饰性悬浮卡片或大面积强渐变。
- 主色只用于关键操作、当前导航、重要进度和少量品牌识别；普通信息使用中性色。
- 所有尺寸优先使用本规范中的值。确需新增尺寸时，应落在 4px 基础网格上。
- 中文界面以可读性为先，正文不得小于 14px，辅助信息不得小于 12px。

## 2. 配色

### 2.1 品牌与功能色

| 令牌 | 色值 | 用途 |
| --- | --- | --- |
| `--color-primary` | `#C81E24` | 主按钮、当前导航、关键链接、核心进度 |
| `--color-primary-hover` | `#A9161B` | 主色悬停状态 |
| `--color-primary-active` | `#871116` | 主色按下状态 |
| `--color-primary-soft` | `#FBE3E4` | 当前导航背景、选中项背景、浅色标签 |
| `--color-brand-deep` | `#3A080B` | 品牌标识、深色强调、深色进度节点 |
| `--color-accent` | `#FF9D2E` | 学习进度、时间轴、提醒和数据亮点 |
| `--color-accent-soft` | `#FFF0DF` | 橙色提示背景、进行中任务背景 |
| `--color-success` | `#278A57` | 完成、在线、成功状态 |
| `--color-warning` | `#D97706` | 警告、待处理状态 |
| `--color-danger` | `#C62828` | 错误、删除、失败状态 |
| `--color-info` | `#3973B9` | 普通系统提示、信息状态 |

### 2.2 背景与边框色

| 令牌 | 色值 | 用途 |
| --- | --- | --- |
| `--color-canvas` | `#F1EFEC` | 应用外层画布、页面最底层背景 |
| `--color-page` | `#FFFFFF` | 主内容区、侧边栏主背景 |
| `--color-surface` | `#FCFBFA` | 卡片和分区的轻微区分背景 |
| `--color-surface-muted` | `#F6F4F2` | 表头、禁用区、次级容器 |
| `--color-border` | `#E8E4E1` | 默认边框、分隔线 |
| `--color-border-strong` | `#D8D2CE` | 输入框、强调分隔线 |
| `--color-overlay` | `rgba(38, 18, 19, 0.42)` | 弹窗遮罩 |

### 2.3 文字色

| 令牌 | 色值 | 用途 |
| --- | --- | --- |
| `--color-text-primary` | `#241719` | 页面标题、正文、关键数字 |
| `--color-text-secondary` | `#655C5D` | 描述、次级正文、导航分组名 |
| `--color-text-tertiary` | `#91898A` | 时间、占位符、辅助信息 |
| `--color-text-inverse` | `#FFFFFF` | 深色或主色背景上的文字 |
| `--color-text-disabled` | `#B8B1B2` | 禁用状态文字 |

### 2.4 配色约束

- 一个页面内主色面积建议不超过可视区域的 12%，橙色点缀不超过 6%。
- 大面积背景只使用 `canvas`、`page`、`surface` 三类中性色。
- 正文与背景对比度至少达到 WCAG AA；禁止在浅橙、浅红背景上使用低对比灰字。
- 渐变仅允许用于独立的数据亮点或计时器模块，推荐 `linear-gradient(135deg, #3A080B 0%, #C81E24 55%, #FFB23F 100%)`，不得作为全页背景。

## 3. 字体

### 3.1 字体族

```css
font-family: "PingFang SC", "Microsoft YaHei", "Noto Sans SC", Arial, sans-serif;
```

数字、计时器和数据指标沿用同一字体族，使用 `font-variant-numeric: tabular-nums` 保持列对齐。英文品牌名可使用 `Arial`，不额外引入装饰字体。

### 3.2 字号层级

| 层级 | 字号 | 字重 | 行高 | 用途 |
| --- | --- | --- | --- | --- |
| 页面主标题 / H1 | `28px` | `700` | `1.3` / `36px` | 每页唯一主标题 |
| 模块标题 / H2 | `20px` | `700` | `1.4` / `28px` | 页面主要分区标题 |
| 卡片标题 / H3 | `16px` | `600` | `1.5` / `24px` | 卡片、面板标题 |
| 正文 | `14px` | `400` | `1.7` / `24px` | 表单说明、内容文本 |
| 强调正文 | `14px` | `600` | `1.7` / `24px` | 导航、按钮、字段名 |
| 辅助文字 | `13px` | `400` | `1.55` / `20px` | 状态、描述、时间 |
| 微型标签 | `12px` | `500` | `1.5` / `18px` | 标签、表头、角标 |
| 核心数据 | `32px` | `700` | `1.2` / `38px` | 统计数字、计时结果 |

### 3.3 字体规则

- 字重只使用 `400`、`500`、`600`、`700`，避免同页出现过多粗细。
- 中文标题不使用全大写、斜体或负字间距；全局 `letter-spacing: 0`。
- 单个面板内最多出现三级字号，靠字重和颜色建立次级对比。
- 移动端 H1 可降至 `24px`，其他正文尺寸保持不变。

## 4. 间距与布局

### 4.1 基础间距

采用 4px 基础网格，常用值如下：

| 令牌 | 数值 | 常见用途 |
| --- | --- | --- |
| `--space-1` | `4px` | 图标与角标、紧凑文字间距 |
| `--space-2` | `8px` | 标签内部、标题与说明 |
| `--space-3` | `12px` | 图标与文字、紧凑控件间距 |
| `--space-4` | `16px` | 表单字段、按钮内边距、小卡片内边距 |
| `--space-5` | `20px` | 卡片内部区块间距 |
| `--space-6` | `24px` | 标准卡片内边距、网格间距 |
| `--space-8` | `32px` | 页面模块间距、桌面端页面边距 |
| `--space-10` | `40px` | 大分区间距 |
| `--space-12` | `48px` | 独立内容章节间距 |

### 4.2 页面结构

- 桌面端采用固定侧边栏加弹性主内容区；侧边栏宽度 `248px`，主内容最大宽度 `1440px`。
- 主内容区桌面端内边距 `32px`，小屏桌面端 `24px`，移动端 `16px`。
- 页面标题区与首个内容模块间距 `32px`；同级模块间距 `24px`；卡片内区块间距 `16px` 或 `20px`。
- 双栏布局默认使用 `minmax(0, 1fr)`，栏间距 `24px`；重要主任务可采用 `3:2` 比例。
- 侧边栏导航项高度 `44px`，相邻项间距 `4px`，导航分组之间间距 `24px`。
- 表单标签与控件间距 `8px`，字段之间 `16px`，表单分组之间 `24px`。

### 4.3 响应式断点

- `>= 1200px`：完整侧边栏，多栏工作台布局。
- `768px - 1199px`：侧边栏可收起，复杂三栏降为两栏。
- `< 768px`：单栏布局，页面边距 `16px`，按钮可占满容器宽度，导航改为抽屉式。

## 5. 组件样式

### 5.1 通用圆角、边框与阴影

| 令牌 | 数值 | 用途 |
| --- | --- | --- |
| `--radius-sm` | `6px` | 标签、图标容器、小控件 |
| `--radius-md` | `8px` | 按钮、输入框、导航项 |
| `--radius-lg` | `12px` | 卡片、面板、弹窗 |
| `--radius-shell` | `16px` | 应用外壳，仅用于大画布展示 |
| `--shadow-card` | `0 4px 16px rgba(58, 8, 11, 0.05)` | 浮起卡片、下拉菜单 |
| `--shadow-modal` | `0 20px 48px rgba(36, 23, 25, 0.16)` | 弹窗、抽屉 |

- 默认卡片使用 `1px solid #E8E4E1`，静态卡片可不加阴影。
- 不允许卡片嵌套卡片；内部信息区使用分隔线、背景色或留白分组。
- 阴影保持柔和低对比，禁止厚重黑影和多层发光效果。

### 5.2 按钮

标准按钮高度 `40px`，水平内边距 `16px`，圆角 `8px`，字号 `14px`，字重 `600`。图标按钮尺寸为 `40px × 40px`，图标建议 `18px`。

| 类型 | 背景 | 文字 | 边框 | 悬停 |
| --- | --- | --- | --- | --- |
| 主按钮 | `#C81E24` | `#FFFFFF` | `#C81E24` | 背景变为 `#A9161B` |
| 次按钮 | `#FFFFFF` | `#241719` | `#D8D2CE` | 背景变为 `#F6F4F2` |
| 轻按钮 | `transparent` | `#655C5D` | `transparent` | 背景变为 `#F6F4F2` |
| 危险按钮 | `#C62828` | `#FFFFFF` | `#C62828` | 背景加深 8% |

- 按钮组间距 `8px`；一个操作区只能有一个主按钮。
- 聚焦状态统一使用 `0 0 0 3px rgba(200, 30, 36, 0.18)`。
- 禁用状态透明度 `0.45`，同时禁用点击和悬停反馈。
- 熟悉的操作优先使用图标或图标加文字；纯图标按钮必须有 `aria-label` 和悬停提示。

### 5.3 卡片与面板

- 默认背景 `#FFFFFF`，边框 `1px solid #E8E4E1`，圆角 `12px`，内边距 `24px`。
- 紧凑数据卡内边距可用 `20px`；移动端统一降为 `16px`。
- 卡片标题与正文间距 `8px`，标题区与主体间距 `20px`。
- 可点击卡片悬停时边框变为 `#D8D2CE`，阴影变为 `--shadow-card`，位移最多 `translateY(-1px)`。
- 统计卡使用细分隔线组织多个指标，不把每个数字再包成独立小卡片。

### 5.4 导航栏

- 侧边栏背景 `#FFFFFF`，右边框 `1px solid #E8E4E1`，不使用大面积深色底。
- 导航文字默认 `#655C5D`，字号 `14px`，字重 `500`；图标 `18px`。
- 当前项背景 `#FBE3E4`，文字与图标 `#C81E24`，圆角 `8px`，字重 `600`。
- 悬停项背景 `#F6F4F2`，文字 `#241719`。
- 品牌区高度至少 `72px`；导航分组标题使用 `12px / 500 / #91898A`。
- 顶部工具栏背景保持透明或白色，高度不做夸张放大，页面标题与操作按钮左右分布。

### 5.5 表单控件

- 输入框、选择框高度 `40px`；多行文本框最小高度 `96px`。
- 背景 `#FFFFFF`，边框 `1px solid #D8D2CE`，圆角 `8px`，水平内边距 `12px`。
- 聚焦时边框 `#C81E24`，外环 `0 0 0 3px rgba(200, 30, 36, 0.12)`。
- 标签使用 `14px / 600 / #241719`；帮助文字使用 `13px / 400 / #91898A`。
- 错误态边框 `#C62828`，错误说明紧邻控件下方，间距 `4px`。

### 5.6 标签与状态

- 标签高度 `24px`，水平内边距 `8px`，圆角 `6px`，字号 `12px`，字重 `500`。
- 默认标签使用 `#F6F4F2` 背景和 `#655C5D` 文字。
- 进行中使用 `#FFF0DF` 背景和 `#B65F00` 文字；完成使用浅绿背景和 `#278A57` 文字；失败使用浅红背景和 `#C62828` 文字。
- 状态不得只依靠颜色表达，应同时提供图标或文字。

## 7. 新页面验收清单

- 已使用本文件规定的颜色变量，没有新增含义重复的色值。
- 标题、正文、辅助文字符合字号和字重层级，正文不小于 14px。
- 页面、模块、卡片和表单间距落在 4px 网格上。
- 按钮、卡片、导航和表单控件符合统一圆角、边框、阴影及交互状态。
- 页面只有一个视觉主操作，主色和橙色点缀没有过度使用。
- 桌面端和移动端均无文字溢出、控件跳动、内容遮挡或横向滚动。
- 交互状态具备默认、悬停、聚焦、按下、禁用及必要的错误反馈。
- 新页面实现前已阅读 `design.md`，实现后已对照本清单完成视觉检查。
