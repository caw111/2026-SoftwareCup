# 第 5 章 评估指标、泛化与错误分析

## 5.1 回归指标

平均绝对误差 MAE 表示预测值与真实值之间绝对误差的平均值，单位与标签一致，容易解释。均方根误差 RMSE 对大误差更敏感，适合关注极端偏差的场景。决定指标前要先理解业务后果。

## 5.2 分类指标

Accuracy 表示预测正确比例，但在类别极不均衡时可能误导。例如 95% 样本都是负类，模型永远预测负类也能得到 95% Accuracy，但对正类没有识别能力。此时应关注 Precision、Recall、F1 或 AUC。

## 5.3 混淆矩阵

混淆矩阵把预测结果拆成 TP、FP、TN、FN，帮助分析错误类型。不同业务对错误类型的容忍度不同。医疗筛查更怕漏诊，垃圾邮件过滤更怕误杀重要邮件。

## 5.4 错误分析

错误分析不是只看平均分，而是找出模型失败的样本群体。可按数据来源、时间段、类别、特征范围或人工标签质量分组。每个错误分析结论都应对应下一步行动。

## 5.5 本章练习

给定一组分类预测结果，计算 Accuracy、Precision、Recall 和 F1。说明只看 Accuracy 会遗漏什么问题。
