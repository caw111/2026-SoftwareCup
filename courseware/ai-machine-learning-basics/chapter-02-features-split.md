# 第 2 章 特征、标签与数据划分

## 2.1 特征与标签

特征是模型用于做判断的输入信息，标签是模型要学习的目标。监督学习的训练过程就是寻找特征与标签之间的统计规律。特征不一定越多越好，错误、重复、泄露或噪声很大的特征会降低模型泛化能力。

## 2.2 训练集、验证集和测试集

训练集用于拟合参数，验证集用于选择方案，测试集只用于最终报告。若反复根据测试集结果调参，测试集就不再代表未知数据，评估结论会过于乐观。

基础划分流程：

1. 先确定预测发生的时间点。
2. 删除预测时不可获得的字段。
3. 按时间或随机方式划分训练集、验证集和测试集。
4. 只在训练集上拟合标准化、缺失值填补等变换。
5. 用相同变换处理验证集和测试集。

## 2.3 分层抽样

分类任务中，如果正负样本比例不均衡，普通随机划分可能导致验证集类别比例失真。分层抽样会尽量保持各集合中类别比例一致，更适合样本不均衡的任务。

## 2.4 数据字段审查表

| 字段名 | 含义 | 预测时可获得 | 缺失情况 | 风险 |
|---|---|---|---|---|
| area | 房屋面积 | 是 | 少量缺失 | 可用 |
| final_price | 成交价格 | 否，作为标签 | 无 | 标签 |
| contract_status | 合同状态 | 否 | 无 | 泄露风险 |
